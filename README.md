# pills trace daily

## introduce

easy usefull program for trace your eating pills record，help remember your daily status

## feature 
- record breakfast，lauch，dinner eating pills time
- the processing showing (at most 3 times daily)
- every meal only one time
- record check
- record edit
- record delte
